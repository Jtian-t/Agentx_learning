# 即将生成的前端代码清单

以下文件将作为第一批前端核心组件生成，旨在实现与 Java 后端的完整闭环：

## 1. 核心布局 (Core Layouts)
- `app/layout.tsx`: 根布局，注入字体与 Toast 容器。
- `app/(auth)/layout.tsx`: 认证页面专用布局，提供居中对齐的卡片背景。

## 2. 页面组件 (Page Components)
- `app/(auth)/login/page.tsx`: 实现登录表单、状态管理及 Token 存储逻辑。
- `app/(auth)/register/page.tsx`: 实现注册表单，包含密码二次确认逻辑。

## 3. 服务层 (Service Layer)
- `lib/api-client.ts`: 封装 `fetch` 请求。
  - 自动注入 `Authorization: Bearer <token>` 请求头。
  - 统一处理 401 认证过期异常，跳转回登录页。

## 4. 类型定义 (Type Definitions)
- `types/auth.ts`: 定义 `User`, `LoginResponse`, `ApiResult` 等 TypeScript 接口。

---

*注：生成代码将优先保证逻辑正确性，UI 样式采用标准的 Tailwind CSS 类名实现。*
