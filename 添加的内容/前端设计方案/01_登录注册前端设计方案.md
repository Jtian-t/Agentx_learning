# AgentX 前端登录注册设计方案 (v1.0)

## 🎨 视觉与交互设计 (UI/UX)
1. **整体风格**: 采用原版 AgentX 的极简现代风格，主色调为深色/白色渐变，强调层次感。
2. **核心容器**: 使用 `max-w-[400px]` 的卡片容器，垂直居中显示。
3. **动态反馈**:
   - **Loading 状态**: 登录/注册按钮在请求期间进入禁用状态并显示加载动画。
   - **Toast 提示**: 登录成功、密码错误、服务器异常等均通过悬浮通知（Toaster）反馈。
4. **响应式**: 完美适配移动端。

## 🛠️ 技术栈选型 (Tech Stack)
- **框架**: Next.js 14+ (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **UI 组件**: Shadcn UI (基于 Radix UI)
- **验证**: Zod + React Hook Form (推荐)
- **图标**: Lucide React

## 🔌 API 交互契约
- **基础路径**: `/api`
- **认证接口**:
  - `POST /login`: 提交 `{account, password}`，返回 `{token, user}`。
  - `POST /register`: 提交 `{email, password, code, nickname}`。
- **Token 存储**: 优先存入 `localStorage` (key: `auth_token`)，并同步写入 Cookie 以支持服务端渲染 (SSR)。
